FHIR-MCP Integration Framework

Transforming legacy EMR systems into FHIR-compliant platforms using the Model Context Protocol (MCP) for AI-powered healthcare workflows while maintaining HIPAA compliance and data security.

View on GitHub Documentation

Project Overview

This project modernizes the CarePulse EMR system by integrating FHIR R4 standards and MCP protocol, enabling AI-powered healthcare workflows.

Legacy EMR System

Next.js-based CarePulse application with comprehensive patient management capabilities and modern UI/UX.

FHIR Integration

Microsoft FHIR Server implementation for R4 compliance with SQL persistence and healthcare interoperability.

MCP Gateway

Helios-based connector enabling seamless AI-EMR communication for intelligent healthcare workflows.
